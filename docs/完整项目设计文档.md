## 1. 项目概述

### 1.1 项目背景
BOM 表图像管理系统是一个基于 GD&T（几何尺寸与公差）标准的企业级尺寸添加及管理系统，旨在为制造企业提供高效、规范、符合行业标准的尺寸管理解决方案。

### 1.2 核心价值
- **省时**：比传统方式节省 50% 的时间
- **省心**：一站式解决方案，无需切换多个工具
- **省力**：简洁直观，低学习成本
- **标准**：完全符合 ISO 1101/ASME Y14.5 和 GB/T 1182 标准

### 1.3 技术栈
- **后端**：Flask + SQLite + SQLAlchemy + Dependency Injector
- **前端**：React 18+ + TypeScript + Tailwind CSS + Zustand
- **API通信**：Axios
- **开发工具**：Vite
- **图像处理**：Canvas
- **Excel处理**：OpenPyXL

### 1.4 遵循的标准
- **ISO 1101**：几何产品规范 (GPS) - 几何公差
- **ASME Y14.5**：尺寸和公差标注标准
- **GB/T 1182**：形状和位置公差
- **GB/T 1804**：一般公差标准
- **ISO 2768**：一般公差标准

## 2. 系统架构设计

### 2.1 整体架构
系统采用前后端分离的 RESTful API 架构：

```
┌─────────────────────────────────┐
│         前端应用层             │
│  React + TypeScript + Tailwind  │
│  Zustand (状态管理)            │
│  Axios (API 客户端)           │
└───────────────┬─────────────────┘
                │ HTTP/REST
                │
┌───────────────┴─────────────────┐
│         后端服务层             │
│  Flask (Web框架)                │
│  依赖注入容器                 │
│  服务接口层 + 仓储层           │
└───────────────┬─────────────────┘
                │
┌───────────────┴─────────────────┐
│         数据持久层             │
│  SQLite (数据库)                 │
│  SQLAlchemy (ORM)            │
└─────────────────────────────────┘
```

### 2.2 后端架构详解

#### 2.2.1 核心组件

1. **服务接口层** (`interfaces/`)
   - 定义业务逻辑接口
   - 实现接口与实现分离
   - 支持依赖注入

2. **仓储实现层** (`repositories/`)
   - 实现数据持久化
   - 封装数据库操作
   - 支持多种数据源

3. **依赖注入容器** (`container.py`)
   - 使用 Dependency Injector
   - 管理服务依赖关系
   - 实现松耦合设计

4. **数据库会话管理** (`database/session.py`)
   - 统一管理数据库连接
   - 实现会话管理
   - 支持事务处理

5. **配置管理** (`config/manager.py`)
   - 环境配置管理
   - 支持多环境配置

6. **标准化API响应** (`api/response.py`)
   - 统一响应格式
   - 统一错误处理

#### 2.2.2 设计模式
- **依赖注入模式**：通过 Dependency Injector 实现松耦合
- **仓储模式**：数据访问与业务逻辑分离
- **服务层模式**：封装核心业务逻辑
- **工厂模式**：对象创建管理

### 2.3 前端架构详解

#### 2.3.1 目录结构
```
src/
├── components/      # UI组件
│   ├── ui/         # 基础UI组件
│   ├── dimension-image-generator.tsx    # 尺寸图片生成器
│   ├── dimension-image-combiner.tsx # 组合图片生成器
│   ├── dimension-image-recognizer.tsx  # 图片尺寸识别器
│   └── ...
├── pages/          # 页面组件
│   ├── projects.tsx      # 项目列表页
│   ├── parts.tsx         # 零件列表页
│   ├── dimensions.tsx      # 尺寸管理页
│   └── ...
├── hooks/          # 自定义Hooks
├── services/       # API服务
│   └── api/
│       ├── axiosInstance.ts
│       ├── projectService.ts
│       └── bomService.ts
├── stores/         # 状态管理
│   └── projectStore.ts
└── types/          # TypeScript类型定义
```

#### 2.3.2 核心组件

1. **DimensionImageGenerator**：尺寸图片生成器
   - 支持普通尺寸绘制
   - 支持几何公差绘制
   - 支持尺寸图片预览
   - 使用 Canvas 绘制

2. **DimensionImageCombiner**：组合图片生成器
   - 支持将同一组的多个尺寸组合成一张图片
   - 自动排版尺寸标注
   - 支持组号管理

3. **DimensionImageRecognizer**：图片尺寸识别器
   - 支持从图片中识别尺寸
   - 识别结果可编辑和删除
   - 批量导入识别结果

4. **TableEditor**：表格编辑器
   - 支持表格编辑
   - 支持批量操作
   - 支持数据验证
