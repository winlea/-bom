## 1. 项目概述

### 1.1 项目背景
BOM 表图像管理系统是一个基于 GD&T（Geometric Dimensioning and Tolerancing，几何尺寸与公差）标准的尺寸添加及管理系统，旨在为制造企业提供高效、规范的尺寸管理解决方案。

### 1.2 核心价值
- **省时**：比传统方式节省 50% 的时间
- **省心**：一站式解决方案，无需切换多个工具
- **省力**：简洁直观，低学习成本

### 1.3 技术栈
- **后端**：
  - Flask Web 框架
  - SQLite 数据库
  - SQLAlchemy ORM
  - OpenPyXL (Excel 导入导出)
- **前端**：
  - React 18+
  - TypeScript
  - Tailwind CSS
  - Zustand (状态管理)
  - Axios (API 通信)
  - Vite (构建工具)

## 4. 后端核心功能模块

### 4.1 项目管理模块

#### 4.1.1 功能概述
项目管理模块负责项目的创建、查询、更新、删除，以及项目信息的完整管理。

#### 4.1.2 核心功能

**1. 项目列表查询**
- **接口**：`GET /api/projects`
- **功能**：获取所有项目列表，按创建时间倒序排列
- **返回字段**：
  - `id`: 项目 ID
  - `name`: 项目名称
  - `description`: 项目描述
  - `status`: 项目状态
  - `created_at`: 创建时间
  - `supplier_name`: 供方名称
  - `address`: 街道地址
  - `supplier_code`: 供方代码
  - `customer_name`: 客户名称
  - `customer_purchase`: 客户采购
  - `quality_engineer`: 质量工程师
  - `phone`: 电话
  - `email`: 邮箱

**2. 创建项目**
- **接口**：`POST /api/projects`
- **功能**：创建新项目，支持 JSON 和 FormData 两种格式
- **请求参数**：
  - `name`: 项目名称（必填，唯一）
  - `description`: 项目描述（可选）
  - `supplier_name`: 供方名称（可选）
  - `address`: 街道地址（可选）
  - `supplier_code`: 供方代码（可选）
  - `customer_name`: 客户名称（可选）
  - `customer_purchase`: 客户采购（可选）
  - `quality_engineer`: 质量工程师（可选）
  - `quality_engineer_signature`: 质量工程师签名图片（可选，BLOB 或 File）
  - `phone`: 电话（可选）
  - `email`: 邮箱（可选）
- **验证规则**：
  - 项目名称不能为空
  - 项目名称必须唯一（UNIQUE 约束）
- **事务处理**：使用数据库事务，失败时回滚

**3. 更新项目**
- **接口**：`PUT /api/projects/:pid`
- **功能**：更新指定项目的信息
- **请求参数**：同创建项目
- **验证规则**：
  - 项目必须存在
  - 项目名称不能为空
  - 项目名称必须唯一（除自身外）

**4. 删除项目**
- **接口**：`DELETE /api/projects/:pid`
- **功能**：删除指定项目及其关联的所有数据（级联删除）
- **验证规则**：
  - 项目必须存在
- **级联删除**：删除项目时，自动删除关联的零件和尺寸

**5. 获取项目详情**
- **接口**：`GET /api/projects/:pid`
- **功能**：获取单个项目的详细信息
- **返回字段**：同项目列表

**6. 仪表板统计**
- **接口**：`GET /api/dashboard/summary`
- **功能**：获取系统整体统计数据
- **返回字段**：
  - `project_count`: 项目总数
  - `parts_count`: 零件总数
  - `parts_with_image`: 有图片的零件数
  - `parts_without_image`: 无图片的零件数
  - `last_import_time`: 最后导入时间

**7. 最近项目和导入**
- **接口**：`GET /api/dashboard/recent`
- **功能**：获取最近 5 个项目和最近 5 次导入记录
- **返回字段**：
  - `recent_projects`: 最近项目列表
  - `recent_imports`: 最近导入记录列表

**8. BOM 导入**
- **接口**：`POST /api/projects/:pid/import`
- **功能**：上传 BOM 文件（CSV 或 XLSX）并创建导入日志
- **请求格式**：multipart/form-data
- **请求参数**：
  - `file`: 上传的文件（可选）
  - `mapping`: 字段映射 JSON（可选）
- **处理流程**：
  1. 保存文件到 `uploads/imports/` 目录
  2. 创建 ImportLog 记录
  3. 返回 202 Accepted 状态
  4. 可通过后续接口触发实际处理

**9. 触发导入处理**
- **接口**：`POST /api/imports/:import_id/process`
- **功能**：同步处理指定的导入日志
- **处理流程**：
  1. 查找上传的文件
  2. 根据文件扩展名选择解析器（CSV 或 XLSX）
  3. 解析文件并写入数据库
  4. 更新 ImportLog 的 created_count 和 errors_count
  5. 返回处理结果

### 4.2 尺寸管理模块

#### 4.2.1 功能概述
尺寸管理模块是系统的核心功能，负责尺寸的创建、查询、更新、删除、批量操作、图片生成、组合尺寸等功能。

#### 4.2.2 核心数据模型

**Dimension 模型字段：**
```python
class Dimension(db.Model):
    __tablename__ = 'dimensions'
    
    id = db.Column(db.Integer, primary_key=True)
    project_id = db.Column(db.Integer, db.ForeignKey('projects.id'), nullable=False)
    part_number = db.Column(db.String(255), nullable=False)
    
    # 尺寸基本信息
    dimension_type = db.Column(db.String(50), nullable=False)  # 尺寸类型
    nominal_value = db.Column(db.Decimal(15, 6), nullable=True)  # 名义值
    tolerance_value = db.Column(db.Decimal(15, 6), nullable=True)  # 对称公差值
    upper_tolerance = db.Column(db.Decimal(15, 6), nullable=True)  # 上公差
    lower_tolerance = db.Column(db.Decimal(15, 6), nullable=True)  # 下公差
    
    # 几何公差相关
    datum = db.Column(db.String(255), nullable=True)  # 基准
    characteristic = db.Column(db.String(255), nullable=True)  # 特性
    
    # 组合尺寸
    group_number = db.Column(db.Integer, nullable=True)  # 组号
    image_url = db.Column(db.String(500), nullable=True)  # 图片 URL
    order_index = db.Column(db.Integer, default=0)  # 排序索引
    
    # 时间戳
    created_at = db.Column(db.DateTime, default=datetime.utcnow)
    updated_at = db.Column(db.DateTime, default=datetime.utcnow, onupdate=datetime.utcnow)
```

#### 4.2.3 核心功能

**1. 获取尺寸列表**
- **接口**：`GET /api/dimensions/projects/:projectId`
- **查询参数**：

- **功能**：获取指定项目（和零件）的所有尺寸

**2. 创建单个尺寸**
- **接口**：`POST /api/dimensions/projects/:projectId`
- **功能**：创建单个尺寸
- **请求参数**：
  - `part_number`: 零件编号（必填）
  - `dimension_type`: 尺寸类型（必填）
  - `nominal_value`: 名义值（可选）
  - `tolerance_value`: 公差值（可选）
  - `upper_tolerance`: 上公差（可选）
  - `lower_tolerance`: 下公差（可选）
  - `datum`: 基准（可选）
  - `characteristic`: 特性（可选）
  - `group_number`: 组号（可选）
  - `image_url`: 图片 URL（可选）
  - `order_index`: 排序索引（可选）
- **验证规则**：

```python
def validate_dimension_group(dimensions: List[Dimension]) -> Tuple[bool, List[str]]:
    """
    验证尺寸组合是否合理
    规则：
    1. 一个组内只能有一个孔径尺寸
    2. 一个孔径可以对应多个位置度
    3. 位置度必须指定基准
    """
    errors = []
    
    # 按组号分组
    groups = {}
    for dim in dimensions:
        group_no = dim.group_number or 0
        if group_no not in groups:
            groups[group_no] = []
        groups[group_no].append(dim)
    
    # 验证每个组
    for group_no, group_dims in groups.items():
        # 检查孔径数量
        diameter_count = sum(1 for d in group_dims if d.dimension_type == 'diameter')
        if diameter_count > 1:
            errors.append(f"组 {group_no} 内有 {diameter_count} 个孔径尺寸，最多只能有 1 个")
        
        # 检查位置度的基准
        for dim in group_dims:
            if dim.dimension_type == 'position' and not dim.datum:
                errors.append(f"尺寸 {dim.id} 是位置度，必须指定基准")
    
    return len(errors) == 0, errors
```

#### 4.2.5 智能检测服务

**1. 尺寸类型自动检测**
```python
def detect_dimension_type(nominal_value: str, notes: str = '') -> str:
    """
    根据尺寸值和备注智能检测尺寸类型
    """
    notes_lower = notes.lower() if notes else ''
    
    # 检查特殊关键词
    if any(keyword in notes_lower for keyword in ['位置', 'position', 'pos']):
        return 'position'
    if any(keyword in notes_lower for keyword in ['同轴', '同心', 'concentric']):
        return 'concentricity'
    if any(keyword in notes_lower for keyword in ['垂直', 'perpendicular']):
        return 'perpendicularity'
    if any(keyword in notes_lower for keyword in ['平行', 'parallel']):
        return 'parallelism'
    if any(keyword in notes_lower for keyword in ['跳动', 'runout']):
        return 'circular_runout'
    
    # 检查名义值格式
    if nominal_value:
        if nominal_value.startswith('R') or nominal_value.startswith('r'):
            return 'radius'
        if nominal_value.startswith('SR') or nominal_value.startswith('sr'):
            return 'sphere_radius'
        if nominal_value.startswith('S⌀') or nominal_value.startswith('SØ'):
            return 'sphere_diameter'
        if '⌀' in nominal_value or 'Ø' in nominal_value or 'ø' in nominal_value:
            return 'diameter'
    
    # 默认返回普通尺寸
    return 'normal'
```

**2. 公差自动计算**
```python
def calculate_tolerance(nominal_value: Decimal, 
                       precision: str = 'medium',
                       standard: str = 'GB/T 1804') -> Tuple[Decimal, Decimal]:
    """
    根据名义值和精度等级自动计算公差
    支持标准：
    - GB/T 1804: 中国国家标准
    - ISO 2768: 国际标准
    """
    # 精度等级对应的公差表（示例）
    tolerance_table = {
        'fine': Decimal('0.05'),
        'medium': Decimal('0.1'),
        'coarse': Decimal('0.2'),
        'very_coarse': Decimal('0.5')
    }
    
    tol = tolerance_table.get(precision, Decimal('0.1'))
    
    # 根据名义值范围调整公差
    if abs(nominal_value) <= Decimal('6'):
        tol *= Decimal('0.8')
    elif abs(nominal_value) <= Decimal('30'):
        tol *= Decimal('1.0')
    elif abs(nominal_value) <= Decimal('120'):
        tol *= Decimal('1.2')
    else:
        tol *= Decimal('1.5')
    
    return tol, -tol
```

const measure = (
  ctx: CanvasRenderingContext2D,
  text: string,
  fontSize: number,
  fontWeight: 'normal' | 'bold'
): TextMeasure => {
  ctx.font = `${fontWeight} ${fontSize}px "Arial", "Microsoft YaHei", sans-serif`;
  const metrics = ctx.measureText(text);
  return {
    width: metrics.width,
    ascent: metrics.actualBoundingBoxAscent || fontSize * 0.8,
    descent: metrics.actualBoundingBoxDescent || fontSize * 0.2,

};

const drawLinearDimension = (
  ctx: CanvasRenderingContext2D,
  symbol: string,
  nominal: string,
  tolText: string | null,
  params: DrawingParams
) => {
  const { startX, startY, mainFontSize, tolFontSize, paddingX, paddingY, gapX } = params;
  
  // 测量文本
  const mMain = measure(ctx, `${symbol}${nominal}`, mainFontSize, 'bold');
  let mTol: TextMeasure | null = null;
  if (tolText) {
    mTol = measure(ctx, tolText, tolFontSize, 'normal');
  }
  
  // 绘制名义值
  setFont(ctx, mainFontSize, 'bold');
  ctx.textAlign = 'left';
  ctx.textBaseline = 'middle';
  const mainBaseline = startY + paddingY + mMain.ascent;
  ctx.fillText(`${symbol}${nominal}`, startX + paddingX, mainBaseline);
  
  // 绘制公差（如果有）
  if (mTol && tolText) {
    const tolX = startX + paddingX + mMain.width + gapX;
    // 公差文本居中对齐
    const tolBaseline = startY + paddingY + Math.max(mMain.ascent, mTol.ascent);
    setFont(ctx, tolFontSize, 'normal');
    ctx.fillText(tolText, tolX, tolBaseline);
  }
};

  // 公差框

  setFont(ctx, 12, 'normal');

  x += toleranceBoxWidth;
  
  // 基准框

**4. 公差文本构建**
```typescript
const buildToleranceText = (
  upperTolerance?: string,
  lowerTolerance?: string
): string | null => {
  if (!upperTolerance && !lowerTolerance) return null;
  
  const ut = upperTolerance || '';
  const lt = lowerTolerance || '';
  
  // 对称公差：+0.1/-0.1 → ±0.1
  if (ut && lt && ut === lt.replace('-', '')) {
    return `±${ut}`;
  }
  
  // 不对称公差
  let result = '';
  if (ut) {
    result += ut.startsWith('+') || ut.startsWith('-') ? ut : `+${ut}`;
  }
  if (lt) {
    if (result) result += '/';
    result += lt.startsWith('+') || lt.startsWith('-') ? lt : lt;
  }
  
  return result || null;
};
```

### 8.3 尺寸数据分组

**前端分组逻辑：**
```typescript
// 将尺寸按组号分组
const groupedDimensions = useMemo(() => {
  const groups: { [key: number]: DimRow[] } = {};
  
  dimensionData.forEach(dim => {
    const groupNo = dim.groupNo || 0;
    if (!groups[groupNo]) {
      groups[groupNo] = [];
    }
    groups[groupNo].push(dim);
  });
  
  // 转换为数组并排序
  return Object.entries(groups)
    .map(([groupNumber, rows]) => ({
      groupNumber: parseInt(groupNumber),
      rows: rows.sort((a, b) => a.id - b.id)
    }))
    .sort((a, b) => a.groupNumber - b.groupNumber);
}, [dimensionData]);
```
